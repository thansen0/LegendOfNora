# Legend of Nora

This is a platformer game, where a player attempts to avoid obstacles and collect items while running along a floor. The player, Nora, runs through the Congo jungle collecting books and avoiding monkeys and other entities. It is written with raylib and C++23.

Comment as needed.

## Project Structure

All code exists in src, and it should be split between the different folders.

Levels and cutscene logic should be placed in src/levels

Characters should be placed in either the player folder or entities folder.

Collectable items (such as the books) should be placed in the items folder.

If you feel a class doesn't fit an existing folder please make a new one.
